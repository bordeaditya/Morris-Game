Project-Morris Game is implemented using Java.
Following Java files are included in the project :
1)ABGame.java
2)ABOpening.java
3)MiniMaxGame.java
4)MiniMaxGameBlack.java
5)MiniMaxGameImproved.java
6)MiniMaxOpening.java
7)MiniMaxOpeningBlack.java
8)MiniMaxOpeningImproved.java
9)MorrisBoard.java

1) Compiling the Project :

To Compile java file:
C:> javac <fileName.java> 

eg :  C:> javac ABGame.java


2) Running the Project :
C:> java <fileName> <path_Board1.txt> <path_Board2.txt> <Depth>

eg: C:> java ABGame C:\\Board1.txt C:\\Board2.txt 3
